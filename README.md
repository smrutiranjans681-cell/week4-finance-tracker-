# Personal Finance Tracker

A modular Python console application for tracking expenses, managing budgets, and exporting reports.

## Features
- Persistence using JSON storage with context management.
- Comprehensive CSV Export and Backup/Restore pipelines.
- Data validation for dates, monetary values, and input safety.
- Visual CLI reports with text-based bar charts.
